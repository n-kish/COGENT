## 🛠️ Setup
Let's start with python 3.10. It's recommend to create a `conda` env:

### Create a new conda environment 
```bash
conda create -n cogent python=3.10
conda activate cogent
```

### Install dependencies
```bash
pip install -r requirements.txt
```
